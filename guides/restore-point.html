How to Create a Restore Point

How to Create a Restore Point in Windows 10 / 11

A restore point lets you return your computer to an earlier working state if something goes wrong.

Step 1: Open System Protection

  1. Press Windows key + S.
  2. Type Create a restore point.
  3. Click on it.

Step 2: Turn On Protection (If Needed)

  1. In the System Protection tab, look under Protection Settings.
  2. Select your system drive (usually C:).
  3. Click Configure.
  4. Select Turn on system protection.
  5. Click OK.

Step 3: Create the Restore Point

  1. Click Create.
  2. Type a name (for example: Before installing driver).
  3. Click Create again.
  4. Wait until Windows confirms it was created successfully.
  5. Click Close.

That’s it ✅

How to Use a Restore Point

  1. Open Create a restore point again.
  2. Click System Restore.
  3. Click Next.
  4. Select a restore point.
  5. Click Next → Finish.
  6. Your PC will restart and restore.

If It Doesn’t Work

Restore Button Is Greyed Out

  • Make sure System Protection is turned on (see Step 2).

Restore Failed

  • Try running System Restore in Safe Mode.
  • Temporarily disable your antivirus.
  • Make sure you have enough free disk space.

No Restore Points Available

  • System Protection may not have been enabled before.
  • Turn it on now to create restore points for the future.
