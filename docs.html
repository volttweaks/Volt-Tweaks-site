Common Fixes

100% CPU Usage

Explanation: CPU Idle is disabled — your CPU is not actually at 100%, this is a visual glitch with Task Manager. If you are air cooled, use the batch file to re-enable CPU idle because it can cause overheating. If you are liquid cooled, leave this on — it gives much lower delay/latency and better overall performance.

App Not Launching / Exiting After a Few Seconds

Make sure you have these installed first:

  • .NET Framework 4.8
  • Visual C++ Redistributable

Try executing these commands — open CMD as admin and paste each line:

cd c:\windows\system32
lodctr /R
cd c:\windows\sysWOW64
lodctr /R
WINMGMT.EXE /RESYNCPERF

Find the Windows Management Instrumentation service, set startup to Automatic and start it.

Video: Windows Management Instrumentation Tutorial
Note: Your WiFi provider may be blocking our authentication servers. Use Cloudflare WARP VPN only while inserting your key — you can uninstall it after.
